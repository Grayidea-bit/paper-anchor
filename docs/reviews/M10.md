## 審查確認（無需變更）

- **座標一致性**：`selectionBBox.ts` 換算 `(rect − holderRect)/scale`，`PageCanvas` 渲染 `x0 × scale`，兩者精確互逆；`data-page`/`data-scale` 由 render 定案值提供，zoom-invariant（E2E 150% 驗證）。
- **RightPane keep-mounted**：ChatPane 常駐渲染，切籤僅用 `.hidden { display:none }` 隱藏，從不 unmount → 串流不中斷（設計正確）。
- **疑點二**（create 失敗只 console.error）：v1 可接受。underline/highlight 無 popover；note 只在 bbox 有效時才呼叫 create，唯一失敗剩真正伺服器錯誤（罕見），此時關閉 popover 尚可接受。
- **list_annotations 工具**：scope-aware（document/project），`list_annotations_scoped` SQL 層 `d.user_id` 恆過濾 + document/project 隔離（比照 `similar_chunks_scoped`），f-string 僅拼內部 filter 片段、全用 bound params，無注入風險；輸出帶 `[C{id}]` + metadata chunks，維持引用鏈。
- **NotesPane**：jumpTo 復用 citation 高亮 store action（同一錨點鏈）；edit/delete `stopPropagation` 不誤觸跳轉。
