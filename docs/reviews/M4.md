## 架構決策回顧（無需變更）

- **RAG 手寫不用框架**：驗證正確——全部檢索+prompt 邏輯 <300 行，除錯時每一步可見。
- **引用協定 `[C{chunk_index}]`**：評測 15/15，模型遵循度好；後端正則容錯（大小寫）夠用。
- **導讀 async 後補**：ready 不被 LLM 延遲卡住，UX 正確。
- **移除 StrictMode**：pdf.js canvas 與 effect 雙跑本質衝突，移除是務實解；若日後想恢復，需把渲染改為 ref-based 單例管理。
